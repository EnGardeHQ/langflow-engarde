Fix "Could not validate credentials" error
Why is this needed?
The JWT secret key was updated on the backend. All existing tokens were created with the old secret and are now invalid. Clearing them will force a fresh login with the new secret.
What this tool does:
  1. Checks for stored authentication tokens
  2. Removes tokens from localStorage
  3. Clears session cookies (if any)
  4. Shows you what was cleared
